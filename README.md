# poopFPS: WebStrike 🌴🔫
 
> A browser FPS that runs entirely in one `index.html` file, held together by `Math.random()` and spite.
 
Ten levels. Bots that occasionally develop object permanence. A jungle so dense you'll question your life choices before you question your K/D. Built with Three.js, several energy drinks, and the emotional resilience to keep debugging raycasting at 2 AM.
 
## Play it
 
Open `index.html` in a browser. That's it. That's the install process. No `npm install`, no 40 dependencies fighting each other, no build step. Just vibes and WebGL.
 
```
double-click index.html
```
 
Wow, look at you, already a power user.
 
## What's actually in here
 
- **10-level bot campaign** with 3 lives per run, because dying twice wasn't humbling enough
- **5 weapons** (pistol, SMG, rifle, shotgun, sniper) each with its own recoil pattern, so you can suck in five uniquely specific ways
- **Grenades**: frag, flash, molotov, smoke — for when "walk up and shoot" feels too honest
- **A shrinking safe zone** because peace was never an option
- **Minimap radar, killstreaks, an ultimate ability**, and enough jungle foliage to make a botanist emotional
- **Procedural audio** — every gunshot, footstep, and death groan is synthesized live, no audio files, no shame
